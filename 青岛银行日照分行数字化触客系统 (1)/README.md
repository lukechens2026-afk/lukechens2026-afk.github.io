# 青岛银行日照分行 - 数字化触客原型系统 (BQD Prototype)

这是一套专为青岛银行日照分行普惠金融部设计的数字化触客移动工作台原型。

## 🚀 项目特性
- **商机管理**：支持商机池查看与分类，快速进入访谈流程。
- **现场访谈画像**：六大维度（基础、经营、供销、生产、融资、需求）结构化数据采集。
- **智能模拟 OCR**：演示环境下的证照识别自动填充功能。
- **智能分析报告**：基于采集数据自动生成企业健康度评分及产品匹配建议。
- **银行级视觉**：采用青岛银行品牌视觉规范，沉浸式 UI 设计。

## 🛠 技术栈
- **React 19**：现代前端框架。
- **Tailwind CSS**：原子化样式引擎。
- **ES Modules (ESM)**：无需构建工具，浏览器原生支持。
- **React Router**：单页应用路由管理。

## 📦 如何部署到 GitHub Pages
1. 将本项目所有文件上传至 GitHub 仓库根目录。
2. 进入仓库 **Settings > Pages**。
3. 选择 `main` 分支并保存。
4. 等待 1-2 分钟即可通过生成的链接进行访问。
